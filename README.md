# The Food District

The Food District is a restaurant website being built using the MERN stack.
This project is currently under development.

## Current Status
- Frontend development in progress
- Home page UI in progress
- Backend development has not started yet

## Upcoming Features
- User authentication
- Dynamic food menu
- Cart and order management
- Admin dashboard
- Backend REST APIs

## Tech Stack
**Frontend**
- React
- HTML
- CSS
- JavaScript
